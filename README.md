# Sword Survival

<img width="1202" height="702" alt="Screenshot 2025-08-08 165934" src="https://github.com/user-attachments/assets/5f8ded50-30bc-4f0b-9571-f9adc8211424" />


A hypercasual 2D sword combat game developed in Godot. Players must survive endless waves of enemies, defeat them using sword attacks, and achieve the highest score possible.

## Gameplay

- Fight waves of enemies
- Attack using a sword combat system
- Score points by defeating enemies
- Survive as long as possible
- Increasing challenge as enemies continue spawning

## Features

- 2D combat system
- Enemy AI movement
- Score tracking
- Wave-based gameplay
- Game Over system
- Sound effects and visual feedback

## Made With

- Godot Engine
- GDScript

## What I Learned

During this project I practiced:

- State-based character actions
- Combat and hit detection
- Enemy spawning systems
- UI implementation
- Scene organization in Godot
- Debugging gameplay mechanics
